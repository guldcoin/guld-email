Email OpenPGP instructions

OpenPGP is the most widely used email encryption standard. It is defined by the OpenPGP Working Group of the Internet Engineering Task Force (IETF) as a Proposed Standard in RFC 4880. OpenPGP was originally derived from the PGP software, created by Phil Zimmermann. Read more.

Linux instructions

  1. Install gnupg
  2. Install evolution (preferable) or Thunderbird + enigmail
  3. Create a pgp key and associate it with your email plugin
  4. Install gpa (optional)

MacOS instructions

  1. Install gnupg
  2. Install evolution (preferable) or Thunderbird + enigmail
  3. Create a pgp key and associate it with your email plugin
  4. Install gpa (optional)
